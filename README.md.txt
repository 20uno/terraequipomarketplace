# Terraequipo Marketplace

Terraequipo es un marketplace especializado en maquinaria pesada que conecta dueños de equipos, operadores y compradores en México.

## Características principales

- Marketplace de renta y venta de maquinaria pesada
- Bolsa de trabajo para operadores
- Cotizador de seguros y fletes
- Tienda de refacciones
- Blog educativo del sector

## Tecnologías utilizadas

- HTML5, CSS3, JavaScript
- Diseño responsivo (mobile-first)
- Carousel para equipos destacados
- Formulario de cotización funcional

Notes on Implementation
This is a front-end implementation only. For a full marketplace, you would need:

Backend API (Node.js, Python, PHP, etc.)

Database (PostgreSQL, MongoDB)

Authentication system

Payment processing integration

The calculator provides a simple demo of the insurance and shipping cost calculation.

The carousels are implemented with basic JavaScript for demonstration purposes.

For a production site, you would want to:

Add proper error handling

Implement server-side validation

Add user authentication flows

Connect to a real database

Implement proper search functionality

Add admin dashboard

The design follows the requirements with:

Clean, intuitive interface

Responsive layout

Search functionality

Featured equipment sections

Job listings

Insurance/shipping calculator

Blog section

Testimonials
